## Why Golang?
- 1. Fast compilation
- 2. No dependencies
- 3. No runtime has to installed on the target machine
- 4. static binaries
- 5. statically linked
- 6. fast startup times
- 7. statically typed, general purpose and systems programming language
- 8. Low latency GC (Comparatively)
- 9. Concurrency (CSP)
- 10. Performance
